# Generating charts

**Note:** *for a Product tour and instructional videos please visit:* [vQuery Youtube Channel](https://www.youtube.com/channel/UCrLNHFgHfw3P0eqKlPLpTwQ)

### Overview

vQuery makes it easier to generate charts by providing a no-code interface where one can create charts by providing a small amount of information.

A python script editor is also available as well as the following plotting libraries: Plotly, Matplotlib, Seaborn.
Most charts can be generated by providing the two first parameters only. The histogram chart only needs at least one

The following APIs are available for scripting:

- Matplotlib
- Plotly
- Seaborn
